# make_sound_ffmpeg

## Input

* clips (e.g. `gotdfather/00.00.03-62/00.00.03-62.mp4`)

## Output

* sound file (e.g. `gotdfather/00.00.03-62/00.00.03-62.mp3`)

## Method

Uses ffmpeg.

Applies to all clips in the collections.

## Run if

No sound file already exists.

## Redo (-r)

Supported.

## Filtering (-f)

Supported
